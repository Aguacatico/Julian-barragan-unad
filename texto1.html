ISO 14598

Esta norma establece un marco de trabajo para evaluar la calidad de software, esta norma proporciona métricas y requisitos para realizar la evaluación de un software. Esta norma es utilizada para aplicar los conceptos impuestos en la norma ISO/IEC 9126, esta norma facilita las actividades necesarias para concluir la evaluación de cualquier tipo de Software Esta norma esta dividida en 6 partes las cuales son :

  • ISO/IEC 14598-1 Visión General: Esta parte provee una visión general de el resto de partes y muestra la relación entre la evaluación que tendrá el producto y lo estipulado por la norma ISO/IEC 9126
  • ISO/IEC 14598-2 Planeamiento y Gestión: Contiene las guías y estándares para la evaluación futura que se le realizara al Software.
  • ISO/IEC 14598-3 Proceso para desarolladores : Provee la guía para la revisión cuando el software sigue en paralelo con el desarrollo y podrá ser realizada por parte del desarrollador
  • ISO/IEC 14598-4 Proceso para adquirientes: provee los recursos para aquellas organizaciones las cuales adquieren software anterior mente desarrollados, estos recursos serán necesarios en caso de que se obtendrá un software pre-desarrollado
  • ISO/IEC 14598-5 Proceso para evaluadores : Esta parte de la norma da las pautas a los evaluadores independientes los cuales llevaran a cabo la revisión del software correspondiente.
  • ISO/IEC 14598-6 Documentación de Módulos: Provee las guías para realizar la documentación necesaria para la evaluación de los módulos Esta norma se recomienda para aquellas empresas que desean adquirir un software pre-diseñado es recomendable utilizarla para dar aprobación sobre un software o para elegir entre un conjunto de alternativas de software.
